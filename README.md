# Student Management System

A Python-based application for managing student records using JSON storage.

## Features

- Add Student
- View Students
- Search Student by ID
- Search Student by Name
- Update Student
- Delete Student
- Total Student Count
- Marks Analytics Report
- Input Validation
- JSON Data Storage

## Technologies Used

- Python
- JSON
- Git
- GitHub

## Concepts Used

- Functions
- Dictionaries
- Loops
- CRUD Operations
- Exception Handling
- File Handling
- Data Validation

## How to Run

```bash
python main.py
```
